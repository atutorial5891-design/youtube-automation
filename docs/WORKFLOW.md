# Workflow

## Daily Flow

1. Research and shortlist topics.
2. Generate a script outline with DeepSeek.
3. Generate a polished script with ChatGPT or Claude.
4. Run agent verification and retry if needed.
5. Apply a tone profile and choose the best variation.
6. Fact-check risky claims.
7. Generate narration, imagery, and final video assets.
8. Complete human review before upload.
9. Publish and monitor analytics.

## Stage 1 Focus

Stage 1 is about making this flow possible: environment setup, credentials, configs, skeleton code, and readiness checks.
