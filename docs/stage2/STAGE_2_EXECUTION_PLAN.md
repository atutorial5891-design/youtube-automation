```python
class HybridScriptGenerator:
    def __init__(self):
        self.deepseek = DeepSeekClient()      # Topic generation ONLY
        self.chatgpt = ChatGPTClient()         # Script writing ONLY
        # OR use Claude instead:
        # self.chatgpt = ClaudeClient()
    
    def generate_script(self, category="health"):
        """
        Complete script generation using HYBRID approach
        Cost: ~$0.30 per script
        Quality: 8.5/10
        """
        
        # STEP 1: Topic Generation (DeepSeek - Cheap & Fast)
        print("Step 1: Generating topic with DeepSeek...")
        topic = self.deepseek.generate_topic(
            category=category,
            trendy=True,
            searchable=True
        )
        # Cost: $0.01-0.02
        # Speed: 2 seconds
        
        # STEP 2: Research & Outline (DeepSeek - Analytical)
        print("Step 2: Creating outline with DeepSeek...")
        outline = self.deepseek.create_outline(
            topic=topic,
            content_type=category,
            video_length="short"  # 2-3 minutes
        )
        # Cost: $0.02-0.03
        # Speed: 3 seconds
        
        # STEP 3: Script Writing (ChatGPT/Claude - QUALITY CRITICAL)
        print("Step 3: Writing engaging script with ChatGPT/Claude...")
        script = self.chatgpt.write_script(
            topic=topic,
            outline=outline,
            style="youtube_faceless",
            tone="engaging",
            length="short"
        )
        # Cost: $0.10-0.15
        # Speed: 5 seconds
        # QUALITY: This determines monetization success
        
        # STEP 4: Metadata Capture
        metadata = {
            "topic": topic,
            "category": category,
            "outline": outline,
            "script": script,
            "api_split": {
                "topic_gen": "deepseek",
                "script_gen": "chatgpt"
            }
        }
        
        return {"script": script, "metadata": metadata}
```

**API Allocation - STRICT RULES:**

| Step | API | Why | Cost |
|------|-----|-----|------|
| Topic Generation | DeepSeek | Analytical, research-based | $0.01 |
| Outline | DeepSeek | Structural, not creative | $0.02 |
| Script Writing | ChatGPT/Claude | CRITICAL for quality | $0.10 |
| NEVER use DeepSeek for script writing | - | Quality <7/10 = fails gates | - |
| NEVER waste ChatGPT on topics | - | Cost wasteful | - |

**Success Criteria:**
- ✅ Topics generated by DeepSeek

**Failure Test 3: API Error Recovery**
```bash
# Test: Simulate API timeout, verify retry with backoff
# Expected: Retries successfully, doesn't crash

python3 << 'EOF'
import time

def api_call_with_retry(max_retries=3):
    for attempt in range(1, max_retries + 1):
        try:
            # Simulate API call
            if attempt < 2:
                # First attempt fails
                raise ConnectionError("API timeout")
            else:
                # Retry succeeds
                return "Success"
        except ConnectionError as e:
            if attempt < max_retries:
                wait_time = 2 ** (attempt - 1)  # Exponential backoff
                print(f"Attempt {attempt} failed, waiting {wait_time}s...")
                time.sleep(wait_time)
            else:
                print(f"Attempt {attempt} failed, giving up")
                raise

# Test the function
try:
    result = api_call_with_retry()
    print(f"Result: {result}")
    print("✅ API error recovery with exponential backoff: VERIFIED")
except Exception as e:
    print(f"❌ API recovery failed: {e}")
EOF
```
