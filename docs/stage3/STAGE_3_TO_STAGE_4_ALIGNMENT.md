## 🔄 DATA FLOW: Stage 3 → Stage 4

### **Complete Handoff Process**

```
Stage 3 Completion
    ↓
[Validate GO Decision]
    ├─ decision = "GO"? → YES
    ├─ ready_for_stage_4 = true? → YES
    └─ All gates passed? → YES
    ↓
[Collect Approved Videos]
    ├─ Load logs/stage_3_final_report.json
    ├─ Load logs/stage_3_approved_videos.json
    └─ Verify 4+ videos approved
    ↓
[Prepare Metadata]
    ├─ Update metadata with Stage 3 results
    ├─ Add YouTube fields
    └─ Verify all required fields present
    ↓
[Quality Assurance Check]
    ├─ Verify video files exist
    ├─ Verify metadata valid JSON
    ├─ Verify qa_score ≥ 75
    └─ Verify no policy violations flagged
    ↓
[Stage 4 Handoff]
    ├─ Copy approved videos to Stage 4 input
    ├─ Copy metadata to Stage 4 input
    ├─ Pass GO decision
    └─ Begin YouTube upload process
```

## 📊 SUCCESS CRITERIA FOR STAGE 3→4 TRANSITION

**Stage 3 Must Deliver:**
- ✅ At least 4/5 videos approved (80%+ approval rate)
- ✅ Gate 3 accuracy 75-85% (fact-check)
- ✅ Gate 4 approval ≥80% (human review)
- ✅ All metadata complete with YouTube fields
- ✅ Quality scores ≥75 for all approved videos
- ✅ No policy violations detected
- ✅ Approved videos manifest created
- ✅ Traceability back to Stage 2 complete

**If Stage 3 Fails:**
- ❌ Decision = "NO-GO" → Return to Stage 2
- ❌ <80% approval rate → Regenerate rejected videos
- ❌ Gate 3 or 4 failed → Fix and re-run Stage 3
- ❌ Policy violations detected → Regenerate problematic sections

## 🎯 STAGE 4 IMPLEMENTATION ROADMAP

### **Stage 4 Will:**
1. **Validate Stage 3 Output**
   - Check GO decision exists and is valid
   - Verify approved videos manifest
   - Validate metadata structure

2. **Prepare for Upload**
   - Generate thumbnails
   - Create upload queue
   - Prepare YouTube API credentials

3. **Execute Upload**
   - Upload video to YouTube
   - Set title, description, tags
   - Configure video settings (public/private/unlisted)
   - Add to playlists

4. **Verify Upload**
   - Confirm video published
   - Verify visibility
   - Check metadata appears correctly

5. **Log Results**
   - Record YouTube video IDs
   - Track upload timestamps
   - Calculate final success rate
