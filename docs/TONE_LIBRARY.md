# Tone Library

## Core Tones

1. Professional Educational
2. Conversational Storytelling
3. Energetic Motivational
4. Curious Explainer
5. Quick and Direct

## Selection Rules

- Match tones to content type first.
- Randomize within eligible tones to avoid repetition.
- Generate 2-3 variations per chosen tone.
- Log the chosen tone for later analytics.
