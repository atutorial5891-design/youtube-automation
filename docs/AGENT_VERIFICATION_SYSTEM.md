# Agent Verification System

## Criteria

- Clarity score >= 70
- Flow score >= 75
- Engagement score >= 70
- No unresolved safety or policy issues
- Overall score >= 72

## Retry Policy

- Initial draft review
- Up to 3 retries if the script fails
- Reject the topic after the fourth failed attempt

## Feedback Format

Each failed review should explain:

1. Which score(s) failed
2. Why they failed
3. What the next regeneration should improve
